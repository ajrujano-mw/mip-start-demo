Derivation of the simplex algorithm

June 2, 2019 · 4 min read
Aster Santana
CEO and founder of MipWise

Consider the following linear program (LP) in standard form:

min⁡cTxs.t.Ax=bx≥0.\begin{array}{rcl} & \min & c^Tx\\ & \text{s.t.}& Ax = b\\ && x \geq 0. \end{array}​mins.t.​cTxAx=bx≥0.​

where AAA is a m×nm\times nm×n matrix and b,cb,cb,c and xxx are vectors of appropriate length. Suppose that the rows of AAA are linearly independent (which implies that n≤mn\leq mn≤m, i.e., the LP has more variables than constraints).

We rewrite the LP above by decomposing the decision variables between basic and non-basic variables. Let BBB and NNN be the set of indices of the basic and non-basic variables, respectively, where BBB has mmm elements and NNN has n−mn-mn−m elements. Then we write xxx as follows:

x=[xBxN]x = \begin{bmatrix} x_B\\ x_N \end{bmatrix}x=[xB​xN​​]

Similarly we decompose ccc and AAA as follows:

c=[cBcN],A=[ABAN]c = \begin{bmatrix} c_B\\ c_N \end{bmatrix}, \quad A = \begin{bmatrix} A_B & A_N \end{bmatrix}c=[cB​cN​​],A=[AB​​AN​​]

Now we rewrite the problem as follows:

min⁡cBTxB+cNTxNs.t.ABxB+ANxN=bxB,xN≥0.\begin{array}{rcl} & \min & c^T_B x_B + c^T_N x_N\\ & \text{s.t.}& A_B x_B + A_N x_N = b\\ && x_B, x_N \geq 0. \end{array}​mins.t.​cBT​xB​+cNT​xN​AB​xB​+AN​xN​=bxB​,xN​≥0.​

Since we have assumed that the rows of AAA are linearly independent, we know that ABA_BAB​ is an invertible matrix. Therefore, we can rewrite the constraints as follows:

ABxB+ANxN=bABxB=b−ANxNxB=AB−1(b−ANxN)xB=AB−1b−AB−1ANxN.A_B x_B + A_N x_N = b\\ A_B x_B = b - A_N x_N\\ x_B = A_B^{-1}(b - A_N x_N)\\ x_B = A_B^{-1}b - A_B^{-1}A_N x_N.AB​xB​+AN​xN​=bAB​xB​=b−AN​xN​xB​=AB−1​(b−AN​xN​)xB​=AB−1​b−AB−1​AN​xN​.

By using this expression for xBx_BxB​, the objective function can be re-written as:

cBTxB+cNTxN=cBT(AB−1b−AB−1ANxN)+cNTxN=cBTAB−1b−cBTAB−1ANxN+cNTxN=cBTAB−1b−(cBTAB−1AN+cNT)xN=cBTAB−1b+(cNT−cBTAB−1AN)xN.\begin{align*} c^T_B x_B + c^T_N x_N &= c^T_B (A_B^{-1}b - A_B^{-1}A_N x_N) + c^T_N x_N\\ &= c^T_B A_B^{-1}b - c^T_BA_B^{-1}A_N x_N + c^T_N x_N\\ &= c^T_B A_B^{-1}b - (c^T_BA_B^{-1}A_N + c^T_N) x_N\\ &= c^T_B A_B^{-1}b + (c^T_N - c^T_BA_B^{-1}A_N) x_N. \end{align*}cBT​xB​+cNT​xN​​=cBT​(AB−1​b−AB−1​AN​xN​)+cNT​xN​=cBT​AB−1​b−cBT​AB−1​AN​xN​+cNT​xN​=cBT​AB−1​b−(cBT​AB−1​AN​+cNT​)xN​=cBT​AB−1​b+(cNT​−cBT​AB−1​AN​)xN​.​

The problem now becomes:

min⁡cBTAB−1b+(cNT−cBTAB−1AN)xNs.t.xB=AB−1b−AB−1ANxNxB,xN≥0.\begin{array}{rcl} & \min & c^T_B A_B^{-1}b + (c^T_N - c^T_BA_B^{-1}A_N) x_N\\ & \text{s.t.}& x_B = A_B^{-1}b - A_B^{-1}A_N x_N\\ && x_B, x_N \geq 0. \end{array}​mins.t.​cBT​AB−1​b+(cNT​−cBT​AB−1​AN​)xN​xB​=AB−1​b−AB−1​AN​xN​xB​,xN​≥0.​ min⁡cBTAB−1bs.t.xB=AB−1bxB≥0.\begin{array}{rcl} & \min & c^T_B A_B^{-1}b\\ & \text{s.t.}& x_B = A_B^{-1}b\\ && x_B\geq 0. \end{array}​mins.t.​cBT​AB−1​bxB​=AB−1​bxB​≥0.​

At any iteration of the simplex method, the non-basic variables are set to zero, i.e., xN=0x_N=0xN​=0. As a result, the values of the basic variables are given by xB=AB−1bx_B = A_B^{-1}bxB​=AB−1​b. And the objective value is given by cBTAB−1bc^T_B A_B^{-1}bcBT​AB−1​b.

To perform a simplex iteration (which is called a pivoting), we attempt to increase one of the non-basic variables from zero. But which one should we choose?

To answer this question, we need to re-write the problem using summation notation:

min⁡cBTAB−1b+∑j∈N(cj−cBTAB−1Aj)xjs.t.xB=AB−1b−∑j∈NAB−1AjxjxB≥0,xj≥0,∀j∈N.\begin{array}{rcl} & \min & c^T_B A_B^{-1}b + \sum_{j\in N}(c_j - c^T_BA_B^{-1}A^j) x_j\\ & \text{s.t.}& x_B = A_B^{-1}b - \sum_{j\in N}A_B^{-1}A^j x_j\\ && x_B \geq 0, x_j \geq 0, \forall j \in N. \end{array}​mins.t.​cBT​AB−1​b+∑j∈N​(cj​−cBT​AB−1​Aj)xj​xB​=AB−1​b−∑j∈N​AB−1​Ajxj​xB​≥0,xj​≥0,∀j∈N.​

From here, it’s easy to see that it only makes sense to increase from zero a non-basic variable xjx_jxj​ if its coefficient cj−cBTAB−1Ajc_j - c^T_BA_B^{-1}A^jcj​−cBT​AB−1​Aj is negative. This coefficient is called the reduced cost of xjx_jxj​. In fact, if the reduced cost of xjx_jxj​ is negative, then increasing xjx_jxj​ will decrease the objective value, which is desirable since this is a minimization problem. On the other hand, if the reduced cost of every non-basic variable is non-negative, we conclude that the current solution is optimal.

Now that we have a way to decide which non-basic variable can be made a basic variable (i.e., pick one with negative reduced cost), suppose x1x_1x1​ is non-basic and has negative reduced cost. From an optimization perspective, we want to increase x1x_1x1​ as much as possible. What can stop us from increasing x1x_1x1​ indefinitely?

The answer comes from the expression

xB=AB−1b−∑j∈NAB−1Ajxj.x_B = A_B^{-1}b - \sum_{j\in N}A_B^{-1}A^j x_j.xB​=AB−1​b−j∈N∑​AB−1​Ajxj​.

Which is same as

xB=AB−1b−AB−1A1x1x_B = A_B^{-1}b - A_B^{-1}A^1 x_1xB​=AB−1​b−AB−1​A1x1​

since xj=0x_j=0xj​=0 for all j∈Nj\in Nj∈N such that j≠1j\neq 1j=1.

Therefore, if the vector AB−1A1A_B^{-1}A^1AB−1​A1 has any positive component, as we increase x1x_1x1​ the corresponding basic variables on the right-hand-side will decrease, and eventually one of them will reach zero and become a non-basic variable! If we were to keep increasing x1x_1x1​, then some basic variables would become negative, which is not allowed as all variables must be non-negative. This completes one iteration of the simplex method.
